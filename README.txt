# Real-Time Chat Room using Socket.IO

This is a basic real-time chat application built with **Node.js**, **Express**, and **Socket.IO**. It allows users to connect to a common room and exchange messages instantly.

# Features

- Real-time chat functionality
- Broadcast messages to all connected clients
- Simple frontend served with Express
- Logs user connection and disconnection events

# Tech Stack

- Node.js
- Express
- Socket.IO
- HTML/CSS/JS (frontend)
